# Historinhas de Fé

Site estático (página única, HTML autossuficiente).

## Estrutura

- `index.html` — a página completa, com todo o CSS e JavaScript embutidos.

## Como rodar localmente

Basta abrir o `index.html` no navegador. Ou servir com qualquer servidor estático:

```bash
npx serve .
```

## Deploy

Hospedado na [Vercel](https://vercel.com) como site estático — sem build. Cada push na branch `main` publica automaticamente.
